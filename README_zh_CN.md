# 隐藏前缀文档

一款思源笔记插件，可以在文档树中隐藏指定前缀开头的文档。

## 功能特性

- 在文档树中隐藏特定前缀开头的文档
- 可配置的前缀：`_`、`.`、`#`、`@`、`~`
- 默认隐藏 `_`（下划线）开头的文档
- 文档树变化时实时更新
- 支持桌面端和移动端

## 使用方法

1. 从思源集市安装插件
2. 点击顶栏插件图标打开设置
3. 选择需要隐藏的前缀
4. 点击"应用"保存设置

选中的前缀开头的文档和文件夹将立即从文档树中隐藏。

## 使用场景

- 隐藏草稿文档（如 `_草稿笔记`）
- 隐藏系统/配置文件（如 `.配置`）
- 隐藏归档内容（如 `#归档文档`）
- 通过隐藏临时或私有文档来整理工作空间

## 设置选项

打开插件设置配置要隐藏的前缀：

| 前缀 | 示例 |
|------|------|
| `_`（下划线） | `_草稿`、`_临时` |
| `.`（点） | `.配置`、`.私有` |
| `#`（井号） | `#归档`、`#旧文档` |
| `@`（艾特） | `@待办`、`@工作` |
| `~`（波浪号） | `~备份`、`~临时` |

## 安装

从思源集市安装：设置 → 集市 → 下载 → 搜索"隐藏前缀文档"
