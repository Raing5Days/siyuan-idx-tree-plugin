# Hide Prefix Docs

A SiYuan plugin that hides documents with configurable prefixes in the document tree.

## Features

- Hide documents starting with specific prefixes in the document tree
- Configurable prefixes: `_`, `.`, `#`, `@`, `~`
- Default setting hides documents starting with `_` (underscore)
- Real-time updates when document tree changes
- Works on both desktop and mobile

## Usage

1. Install the plugin from SiYuan marketplace
2. Click the plugin icon in the top bar to open settings
3. Select which prefixes you want to hide
4. Click "Apply" to save settings

Documents and folders with the selected prefixes will be hidden from the document tree immediately.

## Use Cases

- Hide draft documents (e.g., `_draft-note`)
- Hide system/config files (e.g., `.config`)
- Hide archived content (e.g., `#archived-doc`)
- Organize your workspace by hiding temporary or private documents

## Settings

Open the plugin settings to configure which prefixes to hide:

| Prefix | Example |
|--------|---------|
| `_` (underscore) | `_draft`, `_temp` |
| `.` (dot) | `.config`, `.private` |
| `#` (hash) | `#archive`, `#old` |
| `@` (at) | `@todo`, `@work` |
| `~` (tilde) | `~backup`, `~temp` |

## Installation

Install from SiYuan marketplace: Settings → Marketplace → Download → Search "Hide Prefix Docs"
